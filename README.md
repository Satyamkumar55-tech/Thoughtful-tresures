# Thoughtful Treasures 🎁

An elegant, modern, and responsive e-commerce landing page for a premium gift hamper brand. Designed with a warm, minimalist aesthetic to provide a delightful shopping experience.

## ✨ Features

- **Hero Banner:** A captivating first look at our collection.
- **Featured Hampers:** Highlighted gift sets for internal browsing.
- **Categorized Gifts:** Sections for birthdays, anniversaries, corporate gifts, and festivals.
- **Testimonials:** Heartwarming feedback from our happy customers.
- **Responsive Design:** Optimized for a seamless experience across desktop and mobile devices.

## 🛠️ Technology Stack

- **Frontend:** HTML5, Vanilla CSS
- **Interactions:** JavaScript
- **Development Tool:** Vite

## 🚀 Getting Started

To run this project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Satyamkumar55-tech/Thoughtful-tresures.git
    cd Thoughtful-tresures
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    npm run build
    ```

## 📂 Project Structure

- `index.html` - Main structure
- `styles.css` - Custom styling with a warm, pastel palette
- `script.js` - Interactive components
- `assets/` - Images and visual content
